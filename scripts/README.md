# Scripts

CLI test scripts and utilities for manual testing and debugging.

## Contents

- `cli-test.ts` - Basic CLI testing script
- `cli-test-combat.ts` - Combat system testing script
- `cli-test-effects.ts` - Effects system testing script

## Usage

```bash
npx ts-node scripts/cli-test.ts
npx ts-node scripts/cli-test-combat.ts
npx ts-node scripts/cli-test-effects.ts
```

Note: These are development/debugging scripts, not production code.
